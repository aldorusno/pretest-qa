Testing made with javascript

How to test API

What we need to install:
-NPM (Node.js)

Install through cmd
@babel/cli
@babel/core
@babel/node
@babel/preset-env
@babel/register
chai
mocha
mochawesome
supertest

After everything installed, 
run with this comand on cmd :
npm test -- --mochawesome

You can change token via setting.js

To check report,
Go to folder mochawesome-report
